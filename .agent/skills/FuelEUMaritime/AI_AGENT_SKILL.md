# AI Agent Usage Guidelines

## Objective
Efficiently use AI agents for:
- Code generation
- Refactoring
- Testing
- Documentation

---

## Prompting Rules
- Be specific and scoped
- Provide context (file, function, purpose)
- Avoid vague prompts

---

## Validation
- Always verify generated code
- Check:
  - Architecture violations
  - Logic correctness
  - Edge cases

---

## Documentation Requirement
Log all:
- Prompts used
- Outputs generated
- Corrections made

---

## Best Practices
- Use one agent per task
- Iterate with refinement prompts
- Combine outputs intelligently
