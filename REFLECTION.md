# What you learned using AI agents:
   This was my first time coding using AI agents. It was a dynamic experience, as I was able to code the entire project in a day. And the code was working. About learnings:
   1. AI agents are very good tool in writing boilerplate code but not in writing complex logic.
   2. Initially when I prompted the Agents to code the project the code was not working, So I planned the flow of the project and then provided the agent skills and rules so that they would have the knowledge of the project and code accordingly.
   3. When I planned the flow of the project the agent was able to code the project successfuly, but some bugges were there which I found. So I learn to use agents as tool we cannot entirely depend on them.

# Efficiency gains vs manual coding:
   The gain in efficiency is to much as I was able to code the project in a day which usually require 2-3 weeks. As agent is present in the terminal debugging the code is also become easy.

# Improvements you’d make next time:
   1. I would be more specfic in providing the information of the task as it makes them more better in coding.
   2. Breaking the task into smaller tasks.
   3. Write the flow, skills, rules and readme to provide the addition descriptions of the projects.
